Connect ServiceM8

Paste your API key

Hermia connects to ServiceM8 using an API key generated from your account. This key is encrypted the moment it's saved and is never shared.

1. In ServiceM8, go to Settings → API Keys (or your Add-on / Developer settings).
2. Generate a new key and copy it.
3. Paste it below.
A valid API key is required
